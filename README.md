Nmap Scanning Lab

**Objective**

This project demonstrates hands-on practice with Nmap, a powerful network scanning tool used in cybersecurity for port discovery and service enumeration.

**Tools Used**

* Nmap
* Localhost scanning environment (192.168.1.41)

** Scans Performed**

1. Basic Port Scan
2. Service Version Detection

**Learning Outcome
**
* Identify open ports
* Detect running services
* Understand network exposure and attack surface
