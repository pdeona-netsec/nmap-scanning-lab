# Service Version Detection Scan

## 🔍 Command Used

nmap -sV 192.168.1.41

## 📘 Purpose

To identify the services running on open ports and their versions.

## 📊 Observation

The scan revealed service names and versions, which can help identify outdated or vulnerable services.

## 🧠 Learning

Attackers use service version info to search for known vulnerabilities. Security analysts must monitor exposed service versions.
